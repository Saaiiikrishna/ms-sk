🔐 Admin User MFA Setup

Scan this QR Code with your Authenticator App

📱 Step-by-Step Instructions:

1. Open Google Authenticator or Microsoft Authenticator on your phone
2. Tap "Add account" or "+" button
3. Choose "Scan QR code" or "Scan barcode"
4. Point your camera at the QR code above
5. Your app will show a 6-digit code that changes every 30 seconds
6. Use this 6-digit code to complete admin user creation!
